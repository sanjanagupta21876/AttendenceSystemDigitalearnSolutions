# Attendance Management System

## Features
- Student Login/Logout
- Excel Sheet Integration
- Attendance Summary
- Flask Backend
- Responsive Dashboard

## Project Structure

attendance-system/
│
├── app.py
├── attendance_data.xlsx
├── requirements.txt
│
├── templates/
│   └── index.html
│
├── static/
│   ├── css/
│   ├── js/
│   └── images/

## Run the Project

### Install dependencies
pip install -r requirements.txt

### Start Flask server
python app.py

### Open browser
http://127.0.0.1:5000
